## About The Project

Watchlist App is an application that allows users to search and view details about movies. Users can easily manage their watchlist and watched movies, making it simple to keep track of what they want to see and what they have already seen. The site is built using React for a modern user interface.

<a href="https://i.ibb.co/Y8LDS6F/watchlist-app.gif" target="_blank"><img src="https://i.ibb.co/VDqHGdc/watchlist-app.gif" alt="blog-app" border="0" /></a>

### Key Use Cases:

- Search and view details of movies.
- Easily add movies to the watchlist and watched list.
- Real time updates.

## Installation

To set up the project locally, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/mr-smoke/watchlist-app.git
   ```

2. Navigate to the project directory:

   ```bash
   cd wathclist-app
   ```

3. Install the [dependencies](#dependencies):

   ```bash
   npm install
   ```

4. Set up environment variables:

   - Create a `.env` file in the root directory and add the necessary environment variables.

   ```bash
   VITE_MOVIE_API_KEY=your_api_key
   ```

5. Start the development server:
   ```bash
   npm run dev
   ```

## Usage

To use the watchlist app, follow these steps:

1. Open your browser and navigate to `http://localhost:5173`.
2. Use the search bar to find movies by title.
3. Click the button and add movies to the watchlist or watched list.

### Example Commands

- To search for a movie:

  1. Type the movie title in the search bar.
  2. View the matching results and select the desired movie.

## Features

- **Search Functionality**: Quickly search for movies by title.
- **Watchlist and Watched List**: Easily add movies to the watchlist and watched list.
- **User-friendly Interface**: Intuitive and easy-to-use interface built with React.
- **Real-time Updates**: Dynamic updates without page reloads.
